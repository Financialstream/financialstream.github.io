# Visual Direction (Planning Only)

## Direction
Premium clean financial operations.

## Style
- White / warm off-white background
- Deep navy or charcoal base
- Calm blue/teal accent
- Clean financial dashboard elements
- Large calm typography
- Generous whitespace
- Professional cards
- Subtle borders and shadows
- Minimal decorative effects
- No cheap gradients
- No excessive icons or emoji
- No fake luxury black-gold style
